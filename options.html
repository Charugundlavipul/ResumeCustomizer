ATS Resume Enhancer — Options

1) LaTeX Resume (Overleaf-compatible)

Paste your entire .tex source here. The tool will edit bullets & skills — no hidden text.

2) Resume Keywords (you actually have)

Comma-separated. These constrain what the model can add so it won’t invent skills (e.g., C++).

3) Gemini API Key

Get a key. Stored with chrome.storage.sync.

4) Supporting LaTeX files (optional)

Add custom class/style/assets required to compile locally (e.g., fed-res.cls, .sty, images). These are stored locally and zipped with your main .tex for compilation.
